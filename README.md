# Canadian Urbex Archive

Canadian Urbex Archive is a community-driven urban exploration mapping and documentation project focused on abandoned, forgotten, and historically significant locations across Canada and beyond.

The goal of this project is to create an organized archive for explorers to discover, document, and preserve locations through photography, mapping, research, and shared exploration experiences.

---

## Features

- Interactive exploration map
- Community-submitted locations
- Exploration research tools
- Expanding coverage across Canada & North America
- Many upcoming features
  
---

## Purpose

Canadian Urbex Archive was created to help organize and preserve information related to urban exploration while building a centralized platform for the exploration community.

This project supports:
- Historical interest
- Exploration research
- Community collaboration
- Documentation and archiving

---

## Safety & Respect

Urban exploration involves real risks.

Always:
- Respect local laws and property
- Avoid vandalism or theft
- Never force entry
- Leave locations unchanged
- Prioritize safety at all times

Take only photos. Leave only footprints.
